Defining Color Value

Colors in CSS most often specifies in the following formats:

  1. A COLOR keyword - like "red" , "green" , "blue" , "transparent" , ect.
  2. A HEX value - like "#ff000" , "#00ff00" ,etc
  3. An RGB value (RED , GREEN, BLUE) -like "rgb(225, 0, 0)"
